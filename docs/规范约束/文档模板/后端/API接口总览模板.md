## 兼容性与维护规则

- 接口新增、删除、路径或字段变更必须同步本文及相关详设。
- 废弃字段保留兼容期、替代字段和调用方影响；没有代码证据不写废弃日期。
- Feign/SDK/前端调用方存在时，必须反向核对请求与响应类型。
- 本文不得记录未落地的设计；规划内容只能进入详设。
