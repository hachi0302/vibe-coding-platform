## 强制规则

1. 新增查询或写入前检索现有 Repository/DAO/Mapper 和实体映射，复用已有字段、转换和分页方式。
2. 表、字段、索引、默认值、精度、时区和空值语义以 schema/迁移/实体证据为准。
3. 已执行的历史迁移不可修改；新迁移按项目版本命名并提供前向与回滚影响说明。
4. 事务边界与异常回滚遵守项目现有框架；跨边界一致性不能靠吞错或假成功实现。
5. 变更前全局检索表/字段引用并同步简化物理模型、API、测试和相关长期文档。
