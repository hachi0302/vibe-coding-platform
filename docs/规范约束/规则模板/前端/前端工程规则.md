## 强制规则

1. 页面按现有路由、布局、状态和请求层接入，不在组件内重复创建客户端或全局能力。
2. 优先复用已有组件、hooks/composables、类型、常量和样式 token；新增公共组件前全局检索。
3. 服务端数据、共享状态、URL 状态和组件本地状态按当前项目边界存放，不为方便随意全局化。
4. API 类型、枚举、权限和错误展示与真实契约一致；未知值不默认为成功或第一个选项。
5. 保持现有视觉语言、交互反馈、键盘/可访问性和响应式规则；不因单一截图破坏已有组件体系。
